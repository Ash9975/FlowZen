# 🚀 FlowZen - AI-Powered Order & Packing Workflow Platform

FlowZen is an AI-powered warehouse and order management platform designed for wholesalers, distributors, vegetable suppliers, packing teams, and small businesses. The platform transforms handwritten notes, WhatsApp orders, and uploaded order images into structured, trackable packing checklists using AI.

### 🌐 Live Demo

**Live Application:** https://flow-zen-omega.vercel.app

### 📌 Problem Statement

Many businesses still manage orders through WhatsApp messages, handwritten lists, and spreadsheets. Manual processing often leads to packing errors, missed items, and inefficient workflows.

FlowZen solves this by automatically converting order documents into actionable packing checklists and providing real-time progress tracking.

---

## ✨ Key Features

### 🤖 AI-Powered Order Processing

* Extracts products, quantities, and units from uploaded images.
* Supports handwritten notes, printed orders, and WhatsApp screenshots.
* Automatically generates structured packing checklists.

### 📋 Smart Checklist Management

* Create, edit, update, and delete checklist items.
* Real-time packing progress tracking.
* Order completion workflow with visual success screens.

### 📦 Order Management

* Create and manage customer orders.
* Track order status (Pending, In Progress, Completed).
* Maintain packing history and recent orders.

### 📊 Dashboard Analytics

* Total Orders
* Active Orders
* Completed Orders
* Progress Tracking
* Recent Order Activity
